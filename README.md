# Chain Guestbook

A Base mini app for short onchain guestbook messages. Wallet addresses can sign one short message and the latest notes are displayed in the app.

## Features

- Base mainnet guestbook contract integration
- Wallet connection and onchain write flow
- Latest onchain notes feed
- Base app metadata and Talent verification meta tag
- Transaction attribution tracking via `utils/track.js`

## Contract

- Address: `0xf41cf25dcc080f9af84cb4189215d0d9dc9232bf`
- Type: `On-chain Social / Guestbook`
